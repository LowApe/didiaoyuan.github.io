Python 是解释型语言 java 是编译型语言

  • 解释型语言，发布相当等于公布源码.
  • 解释型语言，运行时间长
  • 编译型语言，通过编译器编译成机器可以读的语言，源代码不公开.
  • 编译型语言运行时间短，因为它编译成机器可读的
