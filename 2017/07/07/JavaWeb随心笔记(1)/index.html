Q3: WEB中 Servlet < load-on-startup >1</ load-on-startup >标签代表什么意思

  1. 伴随服务器的启动，启动Servlet
  2. 标签值中大于0或等于0的表示伴随服务器的启动，自动启动Servlet
  3. 标签值小于0或者没有标签的表示，选择时再启用
  4. 其中数值表示优先级，在很多需要启动的 Servlet 中数值越小优先级越大
